On Space, you can back-up and eject the data stored in every app you install, or any [Collection](/docs/en/use/your-data/collections) you create.

This is done via **Data Exports**. 

## Creating an Export

<InterfaceTabs>
  <Fragment slot="canvas">

For an App on the Canvas, open the Exports menu by clicking an app's context menu from the Canvas and selcting **App Data**. Then select the "Exports" tab.

Click the "Export" button to create an export of your app's data. This will trigger an export process which will run. 
The last status of this export is indicated by the colored dot on the left side. You can click the "Refresh" button to see the latest status.
A successful status (green) indicates that your export was completed successfully and can be downloaded.

</Fragment>
<Fragment slot="collections">

For a Headless Collection, open the Collections app from the Canvas. Choose a Collection to open, .


</Fragment>
</InterfaceTabs>

## Downloading an Export

You can download an any completed data export by clicking the download button on the right side of the list of exports.

This will download a `.zip` archive to your local machine, containing all of the data associated with the export.

### Data Format

The format of the data in an export

```json
/drive
    /folder
        /file
```

## Deleting an Export

To delete an Export, click the trash icon next to the export. This action is irreversible.
